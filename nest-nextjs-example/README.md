## Description

This is a demo project build with Next.js on the frontend and Nest.js on the backend.

## ℹ️ Quick start

```bash
# Install
 yarn
```

```bash
# start server - watch mode
$ yarn run start:dev
```

Open new terminal

```bash
# start next.js web app
% yarn start:next
```

## Server

```bash
# development
$ yarn run start

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
